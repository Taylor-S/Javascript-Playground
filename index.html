Call Info

booWho(bool);

Check if a value is classified as a boolean primitive. Returns true or false.

convertHTML(str);

Convert the characters &, <, >, " (double quote), and ' (apostrophe), in a string to their corresponding HTML entities.
When testing (") make sure to type (\") with the string.
E.g, convertHTML("Hamburgers < Pizza < Tacos");

pairElement("GCG");

Take each character, get its pair and return the results as a 2d array. Function takes T, A, G, C as arguments
For example, for the input GCG, returns [["G", "C"], ["C","G"],["G", "C"]]

triangleRight(n);

Logs a right angled triangle pointing to the right, height and length dependent on user input (n).
For example triangleRight(6); gives a triangle with a height and length of 6

triangleLeft(n);

Logs a right angled triangle pointing to the right, height and length dependent on user input (n).
For example triangleLeft(6); gives a triangle with a height and length of 6

fizzBuzz (n);

Print all number from 1 to (n), for numbers that are divisible by three print "Fizz".
Numbers that are divisible by 5, print "Buzz"
Numbers divisible by both 5 & 3 , print "Fizz Buzz" whilst the other integers all remain remain tha same.

chessBoard(n);

Creates an (n)X(n) chessboard grid.
E.g. chessBoard(8);

missingLetter(str);

Finds the missing letter of a string which is in alphabetical order.
E.g. missingLetter("abcdefghjklmno");
returns "i".

translatePigLatin(str);

Translates the provided string to pig latin.
Pig Latin takes the first consonant (or consonant cluster) of an English word, moves it to the end of the word and suffixes an "ay". If vowel, adds "way" to end of word
E.g, translatePigLatin("glove");
returns "oveglay"

searchAndReplace(str, search, replace);

Performs a search and replace on a sentence using the arguments provided and returns a new sentence.
E.g, searchAndReplace("A quick brown fox jumped over the lazy dog", "jumped", "leaped"); returns "A quick brown fox leaped over the lazy dog"

smallestCommons(arr);

Finds the smallest common multiple of the provided parameters that can be evenly divided by both, as well as by all sequential numbers in the range between these parameters.
E.g, smallestCommons([5, 1]); returns 60

uniteUnique(arr);

finds all unique values in any number of arrays and creates a new array containing all those unique values
E.g, uniteUnique([1, 3, 2], [5, 2, 1, 4], [2, 1]); returns Array [ 1, 3, 2, 5, 4 ]

spinalCase(str);

Converts a string to spinal case. Spinal case is all-lowercase-words-joined-by-dashes.
spinalCase("This Is Spinal Case"); returns "this-is-spinal-case"

sumFibs(num);

Given a positive integer (num), returns the sum of all odd Fibonacci numbers that are less than or equal to (num).
sumFibs(20) returns 23

sumPrimes(num);

Sums all the prime numbers up to and including the provided number.

findElement(arr, func);

A function that looks through an array (first argument) and returns the first element in the array that passes a truth test (second argument).
findElement([1, 3, 5, 8, 9, 10], function(num){ return num % 2 === 0; });
returns 8
